# fisiopao-tu-bienestar-en-casa

Sitio web de FISIO PAO — Bienestar en casa.

**Diseñado y desarrollado por** Juan Pablo Gutiérrez Díaz, especialista en desarrollo web y marketing digital, creando experiencias digitales ágiles y orientadas a resultados.

## Cómo ejecutar el proyecto

1. Clona el repositorio:

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
```

2. Instala dependencias:

```sh
npm install
```

3. Ejecuta en desarrollo:

```sh
npm run dev
```

4. Para producción:

```sh
npm run build
npm run preview
```

## Tecnologías

- Vite
- TypeScript
- React
- Tailwind CSS
- shadcn-ui
